3、JavaScript 函数的参数和大多数其他语言中的参数有所不同。JavaScript 函数不介意传递进来多少个参数，也不介意参数的数据类型。即使定义的参数只接受两个参数，在调用函数的时候也未必需要传递两个参数。可以选择传递一个、三个甚至不传递。
